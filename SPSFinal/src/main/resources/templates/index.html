How It Works

👤

1. Register / Login

Create a free customer account or log in to access all features and manage your bookings.

🔍

2. Browse Providers

Filter by service type, search by name, and view ratings and reviews from real customers.

📋

3. Book a Service

Choose a verified provider, pick a date, add notes and confirm your booking in seconds.

⭐

4. Rate & Review

After the job is complete, leave a star rating and feedback to help the community.

🛡️

Admin Controls

Admins verify providers, manage bookings and mark jobs complete so ratings can be given.

📡

REST API

Full JSON REST API available at /api/providers for external integrations.
